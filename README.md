# Breast Cancer Tumor Classification: A Comparative Study
This repository contains a comprehensive research and implementation of five Machine Learning algorithms to classify breast cancer tumors (Malignant vs. Benign).

## 🌍 Language Note
**Please note:** The current project documentation and code comments are in **Arabic**. An English translation is currently in progress and will be available soon.

## 🔬 Project Overview
The study evaluates the performance of five core classification algorithms using the Wisconsin Diagnostic Breast Cancer (WDBC) dataset:
- Logistic Regression (LR)
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Decision Trees (DT)
- Naive Bayes (NB)

## 📊 Strategic Findings
- **Technical Winner:** Logistic Regression (LR) achieved the highest accuracy and AUC scores.
- **Expert Choice:** Despite LR's performance, **SVM** is recommended for medical data due to its efficiency in handling non-linear decision boundaries.
- **Clinical Priority:** The study prioritizes minimizing **False Negatives (FN)** to ensure patient safety in medical diagnostics.
